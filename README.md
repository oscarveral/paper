# Paper

Implementations and experiments of academic papers I find interesting. This repository serves as a personal archive and technical portfolio where I translate theoretical concepts from research papers into working code. The goal is to deeply understand the underlying mechanics of these systems by building them from scratch.

## Categories

The implementations are modularized and categorized by domain. Each category contains its own documentation, specific paper references, and implementation details.

* **[Cryptography](cryptography/README.md)**: Implementations of cryptographic primitives, encryption schemes, and related protocols.
